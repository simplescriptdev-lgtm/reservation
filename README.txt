# Table Reservation App (PHP + SQLite/MySQL)

## Швидкий старт (локально, з SQLite)
1) Скопіюй `config.sample.php` у `config.php` (залиш `use_sqlite => true`).
2) Запусти PHP сервер у цій папці:
   ```bash
   php -S localhost:8000
   ```
3) Відкрий `http://localhost:8000/seed.php` один раз — створить таблиці і користувача.
4) Логін: **admin@example.com**, пароль: **admin123**.
5) Переходь на `/dashboard.php`.

## MySQL/MariaDB
1) Створи БД `restaurant` (або іншу).
2) Імпортуй `schema.sql` у свою БД.
3) В `config.php` вистави `use_sqlite => false` і налаштуй `db_dsn`, `db_user`, `db_pass`.
4) Створи користувача вручну або тимчасово увімкни SQLite seed, щоб згенерувати хеш.

## Функціонал MVP
- Авторизація. У шапці показує ім'я залогіненого.
- Флор-план (SVG): створення столів, drag&drop, збереження позицій.
- Панель ліворуч: кількість столів, список, видалення.
- Резервації по днях: таблиця з колонками (час, прізвище, к-сть, стіл, нотатки) та форма додавання.
- API: `/api.php?action=...`

## Наступні кроки
- Перевірка конфліктів резервацій (перетини у часі на одному столі).
- Пошук вільних столів за інтервалом.
- Ролі (адмін/хостес), історія змін.
- Статуси столів: вільний / заброньований / посадка / рахунок.
- Експорт у CSV, друк листа резервацій.
- Мобільна адаптація та перетягування торканням (touch).
