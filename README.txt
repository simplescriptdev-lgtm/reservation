# Table Reservation App — v2 (full)

## Нове
- Кольори столів: вільний — зелений, зайнятий (на обрану дату) — червоний.
- Історія: видалені резервації мають `deleted_at` і відображаються у нижній таблиці.
- Менші іконки столів (~42px).
- Зони: вкладки «Зал / Тераса», фільтр столів на схемі.

## Швидкий старт (SQLite)
1) Переконайся, що існує `config.php` (у цьому архіві він уже є).  
2) Запусти локальний сервер у цій папці:
   ```bash
   php -S localhost:8000
   ```
3) Ініціалізація БД:
   - Нова база: відкрий `http://localhost:8000/seed.php`
   - Оновлення зі старої v1: відкрий `http://localhost:8000/migrate.php`
4) Перейди на `http://localhost:8000/login.php` (логін: admin@example.com / admin123).

## MySQL
- Вимкни SQLite в `config.php` (`'use_sqlite' => false`) і заповни `db_dsn`, `db_user`, `db_pass`.
- Виконай міграції з `migrate.php` (виводить готові SQL).

## Примітки
- Статус «зайнятий» наразі на рівні дати. Можемо додати інтервали часу (наприклад, 2 год за замовчуванням).
